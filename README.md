# web-pantun

Website generator pantun interaktif berbasis Next.js.

## Fitur
- Desain modern, responsif, dan interaktif
- Generator pantun otomatis/interaktif
- Transisi dan animasi halus

## Cara Menjalankan

Pertama, jalankan development server:

```bash
npm run dev
# atau
yarn dev
# atau
pnpm dev
# atau
bun dev
```

Buka [http://localhost:3000](http://localhost:3000) pada browser Anda untuk melihat hasilnya.
